
# 删除分支(1)——删除分支

## 简介

`git-delete-branch` \<鳃鲨>

## 描述

删除本地和远程分支名\<鳃鲨>.   注意,如果签出分支,则本地删除失败.

## 选项

  \<鳃鲨>

要删除的分支的名称.如果提供多个分支,那么它们都将被删除.

## 实例

```
$ git delete-branch integration
$ git delete-branch integration bug/1234
```
