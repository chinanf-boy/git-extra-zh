
# Git解锁(1)——解锁一个排除在版本控制之外的文件

## 简介

`git-lock` \<文件名>

## 描述

从版本控制解锁本地文件.

## 选项

  \<文件名>

文件名的名称.

## 实例

```
$ git unlock config/database.yml
```
