
# 如何:生成文档

## 描述

在生成的文档:

1)"开始由填写人template.md’

(2)然后ronn安装程序.[Get ronn from github.](https://github.com/rtomayko/ronn)

组成:3)运行

```
$ make -C .. man/git-<command>.{1,html}
```

4)记住,我们使用下面的文件命名规则为:

```
git-<command>.html
git-<command>.1
git-<command>.md
```
