
# GIT移植(1)——合并并销毁给定分支

## 简介

`git-graft` \<SRC分部> \<Dead分支>

## 描述

合并提交\<SRC分部>进入之内\<Dead分支>

## 选项

  \<SRC分部>

  \<Dead分支>

## 实例

```
$ git graft new_feature dev
```
