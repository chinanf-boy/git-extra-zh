
# 删除标签(1)——删除标签

## 简介

`git-delete-tag` \<标记名>

## 描述

删除本地和远程标记名\<标记名>.

## 选项

  \<标记名>

要删除的标签的名称.如果提供多个标签,那么它们都将被删除.

## 实例

```
$ git delete-tag 0.0.1
$ git delete-tag 0.0.1 0.0.2
```
