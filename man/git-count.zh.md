
# Git计数(1)-显示提交计数

## 简介

`git-count` [——所有--]

## 描述

显示提交计数.

## 选项

——所有

显示提交计数详细信息.

## 实例

输出提交总数:

```
$ git count

total 1844
```

输出冗长提交计数详细信息:

```
$ git count --all
```
