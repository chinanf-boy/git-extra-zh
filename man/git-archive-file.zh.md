
# git-archive-file(1)——将Git存储库的当前HEAD导为zip压缩包

## 简介

`git-archive-file` 

## 描述

将存储库的当前HEAD导出为，具有可识别和唯一名称的存档.

## 选项

此命令不采取任何选项.

## 实例

归档命名约定:

关于任何分支:`git-extras.1.7.0-110-gafefba7.branch-name.zip`

主分支:`git-extras.1.7.0-110-gafefba7.zip`

在独立HEAD上(例如标签):`git-extras.1.7.0.zip`
