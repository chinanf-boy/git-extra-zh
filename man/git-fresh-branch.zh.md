
# Git新鲜分支(1)——创建新的分支

## 简介

`git-fresh-branch` \<鳃鲨>

## 描述

创建空的本地分支名\<鳃鲨>.

## 选项

  \<鳃鲨>

要创建的分支的名称.

## 实例

```
$ git fresh-branch docs
```
