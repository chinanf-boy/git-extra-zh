
# Git清除(1)——严格清理存储库

## 简介

`git-clear`

## 描述

清除存储库,使其看起来像是用当前的头新克隆的状态.基本上,它是一个git-reset——与删除所有驻留在工作目录内的未跟踪文件(包括那些在.gitignore中的文件)一起很难.

## 实例

清除回购协议.

```
$ git clear
```
